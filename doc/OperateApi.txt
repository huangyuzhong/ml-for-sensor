
/api/rest/operate/device/floor/{deviceId}?type=&floorId=&floorNum=&scientist=&name=&email=&mobile=&productNum=
deviceId：当前设备Id
入参：{
type    1是修改，0是添加
floorId 修改时制定当前的层Id
floorNum    添加的第几层的数量，int值
scientist   科学家名字
name        存放物品名称
email       科学家邮箱
mobile      科学家手机
productNum  实验品数量
}

/api/rest/room/current/data?roomId=
入参：
{
roomId：当前房间Id
}

/api/rest/file/create/device/{name}?roomId=&typeId=&code=&xPoint=&yPoint=&name=         创建设备
{name} 登陆人用户名
入参：{
roomId：     房间Id
typeId：     所选设备种类id
code：       设备编号
xPoint：     所在房间的x坐标
yPoint：     所在房间的y坐标
name：       设备名称
file：       上传的设备图
}

/api/rest/operate/device/{deviceId}?name=&creator=&maintain=&maintainAlterDays=&model=
{deviceId}  设备id
入参：{
name：                   设备名称
creator：                制造商
maintain：               保养周期
maintainAlterDays：      提前提醒天数
model；                  品牌名称
}
